-   题目 1：

```js
var a = { n: 1 };
var b = a;
a.x = a = { n: 2 };

// 请写出以下输出结果
console.log(a.x);
console.log(b);
```

::: details 答案

```js
// undefined
// { n: 1, x: { n: 2 } }
```
